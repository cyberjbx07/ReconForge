# ReconForge 🔥

Advanced Enumeration Framework built using Python.

## 🚀 Features

* DNS Enumeration (A, MX, NS)
* Subdomain Discovery (wordlist-based)
* Port Scanning (Nmap)
* Service & Version Detection
* Risk Analysis Engine (HIGH / MEDIUM / LOW)
* Report Generation (TXT)

## 🧠 Tech Stack

* Python 3.12
* dnspython
* python-nmap
* Nmap

## ⚙️ Setup

```bash
git clone https://github.com/<your-username>/ReconForge.git
cd ReconForge

# Create virtual environment
py -3.12 -m venv venv
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```
